In order for you to use this system in standard view, you need to have JavaScript enabled in your browser. To do so, please follow the instructions below:

Google Chrome (15.0)

  1. Click the spanner icon on the browser toolbar.
  2. Select Options.
  3. Click the Under the Hood tab.
  4. Click Content Settings in the 'Privacy section.'
  5. Select Allow all sites to run JavaScript in the 'JavaScript' section.
  6. Then Refresh(F5) your browser.

Mozilla Firefox (8.0)

  1. Select Tools from the top menu.
  2. Choose Options.
  3. Choose Content from the top navigation.
  4. Select the checkbox next to Enable JavaScript and click OK.
  5. Then Refresh(F5) your browser.

Internet Explorer (9.0)

  1. Select Tools from the top menu.
  2. Choose Internet Options.
  3. Click on the Security tab.
  4. Click on Custom Level.
  5. Scroll down until you see the section labeled 'Scripting.'
  6. Under 'Active Scripting,' select Enable and click OK.
  7. Then Refresh(F5) your browser.

Apple Safari (5.0)

  1. Open the Safari menu on your browser's toolbar.
  2. Choose Preferences.
  3. Choose Security.
  4. Select the checkbox next to Enable JavaScript.
  5. Then Refresh(F5) your browser.

Please keep in mind that upgrading your browser or installing new security software or security patches may affect your JavaScript settings. It is a good idea to double-check that JavaScript is still enabled if you notice problems displaying this System.
